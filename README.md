# to-do-list
⚠️ Em Desenvolvimento ⚠️

To Do List feita em Angular e Node como forma de estudo.

## Rodando front-end
#### Tecnologias
- Node v18.19
- Angular

#### Como rodar
- Entre na pasta front/ e instale as dependências:
`npm i`

- Rode o projeto local:
`npm start`

## Rodando back-end
#### Tecnologias
- Node v18.19
- Express

#### Como rodar
- Entre na pasta back/ e instale as dependências:
`npm i`

- Rode o projeto local:
`npm start`
